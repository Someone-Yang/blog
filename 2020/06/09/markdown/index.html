Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。
我将在此列举Markdown常用语法，供像我一样的入门者参考。

基础语法

一般来说，所有支持Markdown的平台都支持下列语法。

换行(最重要)

现在貌似很多人不会换行XD

句尾空两格或两段之间空行。

我是第一句  
我是第二句

我是第三句

效果：
我是第一句
我是第二句

我是第三句

斜体

*斜体文本*

效果：
斜体文本

加粗

**加粗文本**

效果：
加粗文本

连接

[提示文本](链接)
[含悬浮提示的链接](链接 "提示")

效果：
提示文本
含悬浮提示的链接

图像

![占位符](链接)

效果：

行内代码

`代码`

效果：
代码

区块代码

`(共3个)
代码
`(共3个)

效果： 

代码

列表

* 项目一
* 项目二

效果：

  • 项目一
  • 项目二

标题

# h1
## h2
### h3
#### h4
##### h5
###### h6

引用

> 内容

效果：

内容

水平分割线

---

高级语法

并不是所有支持Markdown的平台都支持这些拓展语法。

表格

|表头1|表头2|表头3|
|:---|:---:|---:|
|左对齐|中对齐|右对齐|

复选框

- [] 未选中
- [x] 选中

高亮

==高亮文本===

脚注

原文[^1]
[^1]:注释
Markdown
