这是什么？

这个可爱的东西叫做“YWCED”(Yang Windows CMD Echo DLL)(雾)，是一个只能用于Windows平台动态链接库(雾x2)，可以用来在控制台上达到某些特殊目的(即zhuang-bi)。

我要怎么用？

下载，头部引用，敲敲代码，运行，崩溃。
